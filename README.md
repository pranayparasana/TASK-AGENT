# 🤖 AI Task Agent — Agentic Workflow System

Autonomous AI agent that breaks down high-level goals into subtasks,
uses tools to complete them, and delivers a structured report.

## Tools Available
- Web Search (DuckDuckGo)
- Calculator
- File Save

## Tech Stack
Python · Groq LLaMA3 · Streamlit · DuckDuckGo Search

## How to Run
1. Clone this repo
2. pip install -r requirements.txt
3. Add your Groq API key to .env: GROQ_API_KEY=your-key
4. streamlit run app.py
